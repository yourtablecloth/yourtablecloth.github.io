# 후원 안내

**식탁보**는 금융·공공 사이트 이용에 필요한 보안 구성 요소를 안전하게 준비하도록 돕는 **무료 공개 소프트웨어**입니다.

식탁보는 광고 없이 제공되며, 아래 유지 항목을 위해 지속적인 비용이 발생합니다. 필요하다고 느끼신다면 유지에 참여하실 수 있습니다.

위 3가지 유지 항목(코드 서명 인증서, 인프라 운영, 호환성 테스트)에 사용됩니다.
